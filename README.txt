COVID-19 Growth rates
=====================

Auto updateable spreadsheet with a pivot table and graphs to compute the growth rates of infections.

The goal to avoid exponential growth is to achieve a growth rate below 1.


Instructions
------------

1. Run update.sh
2. Open the .fods
3. Click on the «enable content» to allow LibreOffice to read the downloaded .csv's
4. Optionally, customize the «config» sheet. Make sure you do not exceed the specified number of rows.
